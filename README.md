# 🛰️ TOS – Asteroid Extraction Directive

**Tournament of Strategies — Arcade Edition (2026)**

A browser-based strategy game where players manage a planetary mining syndicate with a fixed **1,000 Credit** budget. Build mining fleets, bid for launch priority, extract valuable asteroid resources, and maximize profits before your competitors.

---

## 🎮 Features

- 🚀 Fleet-building strategy
- 💰 Budget management
- 🛰️ Asteroid mining simulation
- ⚡ Resource decay mechanics
- 🤖 AI-controlled rival syndicates
- 📊 Live scoring and leaderboard
- 🌐 Runs entirely in the browser

---

## ▶️ Getting Started

Clone the repository:

```bash
git clone https://github.com/Mounees-321/TOS-Asteroid_Extraction.git
cd TOS-Asteroid_Extraction
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:3000
```

---

## 🎯 Gameplay

Each player starts with **1,000 Credits** and must strategically allocate funds to:

- 🚀 Engine Power
- 📦 Cargo Capacity
- 🎫 Launch Agency Bid

Ships compete for limited asteroid resources, where launch order, travel time, cargo size, and resource decay determine profitability.

The winner is the syndicate with the highest total credits after all mining operations are complete.
